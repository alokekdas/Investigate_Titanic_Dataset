""" Draw a pie-chart with the port of embarkation statistics.
    matplotlib provides a convenient way to denote the percentage number.
    However, by default, the plot does not include the actual value for display.
"""

"""
    Function: make_autopct
    Purpose: Customize the ability to display the percentage information in a specific format.
    Input: The values to be plotted (use these values to calculate the sum total).
    Defines: Another function 'pct_and_value' that recieves the percentage number from plt
           and returns back a label to plt
    output: Returns the pointer to the function 'pct_and_value'. This funtion is used by plt to
           display a label on the pie.
    (Courtesey: Suggestion by the reviewer)
"""
def make_autopct(values):
    total = sum(values)
    def pct_and_value(pct):
        val = int(round(pct*total/100.0))
        return '{p:.2f}%  ({v:d})'.format(p=pct,v=val)
    return pct_and_value

labels = 'Cherbourg', 'Queenstown', 'Southampton'
sizes = [passengers_C, passengers_Q, passengers_S]
colors = ['yellowgreen', 'gold', 'lightskyblue']

""" Only "explode" the 2nd slice, i.e. 'Queenstown' passengers """
explode = (0, 0.1, 0)  
plt.pie(sizes, explode=explode, labels=labels, colors=colors,
        autopct=make_autopct(sizes), shadow=True, startangle=90)

plt.title('Passengers by port of embarkation')
""" Set aspect ratio to be equal so that pie is drawn as a circle """
plt.axis('equal')
plt.show()

Limitation¶

The Titanic dataset has some limitations. The missing values in some cases, e.g. age, cabin etc. likely makes the analysis incomplete, to some extent. Based on the analysis, one would tend to infer a correlation between age and the rate of survival. However, based on some missing data, drawing such conclusions could be erroneous. The statistical population (all Titanic passengers) parameters are known in this case. A z-test could possibly be conducted to establish if there is really a relationship between the the age and the survival rate. In absence of such statistical test, we will not make any inferences about the causation.

The dataset did not include the number of lifeboats available. If this number was available, then we could conclude if the boats were sufficient or not for these many passengers.

Conclusion¶

In this project, we have analyzed the Titanic passengers data and tried to find out if there is any insight we can get to correlate to the survival rate. The average and median age of the passengers were about 30. There were a few babies and aged people among the passengers. There were also large families travelling in the ship.

Analysis shows that more than half of the passengers perished. The analysis also shows that there are more female survivors than the male survivors. Besides, the age distribution data shows that more young people died, maybe because they wanted to let the babies and the elderly people be rescued first.

Given some limitation in the dataset, we can observe some pattern for survival rate on the age and gender. However, no statistical test, like z-test, was performed to test any hypothesis.
